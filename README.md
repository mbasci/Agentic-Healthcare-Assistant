# Agentic Healthcare Assistant using Gradio MBA

Capstone project using Gradio for notebook testing and Streamlit for deployment.

## Features
- Chat with Healthcare Agent
- Add Patient Record
- View Patient History
- Summarize Patient History using AI
- Book Appointment
- View Appointments
- Disease Information
- Agent Logs
- Evaluation Metrics

## API Key
This app requires an OpenAI API key.
Do not hardcode the key.
For Streamlit Cloud, add OPENAI_API_KEY under Settings > Secrets.

## Run Locally
pip install -r requirements.txt
streamlit run app.py
