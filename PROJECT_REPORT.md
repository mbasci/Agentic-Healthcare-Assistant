# Project Report\n\n## Project Title\nAgentic Healthcare Assistant using Gradio MBA\n\n## Objective\nBuild a simple agentic healthcare assistant for medical task automation.\n\n## Main Functions\n- Add patient records\n- Retrieve patient history\n- Summarize patient history using AI\n- Book doctor appointments\n- Provide disease information\n- Show agent logs\n- Show simple evaluation metrics\n\n## Agentic Design\nThe assistant uses simple keyword routing to decide which tool to call.\n\n## Safety Controls\nThe assistant is instructed not to diagnose, prescribe, or replace a doctor.\n\n## Limitations\nThis is a demonstration project, not a clinical system.
